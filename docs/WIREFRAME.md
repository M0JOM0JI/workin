## 화면 목록

### 공통
- S-00. 스플래시
- S-01. 온보딩 (역할 선택)
- S-02. 회원가입
- S-03. 로그인

### 오너 (Owner)
- B-00. 온보딩 — 매장 생성 (첫 매장, 매장 없을 때만 표시)
- B-01. 홈 / 대시보드
- B-02. 스케줄 관리 (주간)
- B-03. 스케줄 관리 (월간)
- B-04. 알바생 목록
- B-05. 알바생 상세
- B-06. 초대 (알바생/매니저 구분)
- B-07. 출근 현황 (실시간)
- B-08. 급여 관리 (월별 요약)
- B-09. 급여 명세 (알바생별)
- B-10. 매장 설정 (오너 전용)
- B-11. 매장 추가 모달 (사이드바 버튼)

### 매니저 (Manager)
- M-01. 매니저 합류 플로우 (초대 코드 입력)
- M-02. 매니저 홈 (B-01과 동일 UI, 권한 차이만 있음)
- M-03. 알바생 상세 (매니저 승격 버튼 없음)
- M-04. 알바생 초대 (알바생 역할만 발급)
- M-05. 매장 설정 (읽기 전용)

### 알바생 (Staff)
- S-10. 홈 (출근 체크)
- S-11. 스케줄 확인
- S-12. 출근 기록
- S-13. 급여 확인
- S-14. 프로필 / 설정

### S-00. 스플래시
```
┌─────────────────────┐
│                     │
│                     │
│      [Workin]       │
│       로고/아이콘    │
│                     │
│                     │
└─────────────────────┘

- 앱 실행 시 1.5초 표시 후 자동 이동

> **표시 조건**: 로그인 성공 후 소속 매장이 0개일 때 `/onboarding/store`로 리다이렉트

│                             │
│ 매장명 *                    │
│ [________________________]  │
│                             │
│ 사업자명 *                  │
│ [________________________]  │
│                             │
│ 사업자번호 *                │
│ [000  -  00  -  00000    ]  │
│                             │
│ 주소 *                      │
│ [________________________]  │
│ [상세주소                ]  │
│                             │
│ 매장 전화번호               │
│ [________________________]  │
│                             │
│ 담당자 휴대폰번호           │
│ [________________________]  │
│                             │

**동작**

- 뒤로가기 차단 (매장 없이 대시보드 진입 불가)

### B-11. 매장 추가 모달 (사이드바 버튼)

> **표시 조건**: 매장이 1개 이상 있을 때 사이드바 하단에 `+ 매장 추가` 버튼 항상 노출

**사이드바 매장 영역 (변경 후)**
```
┌────────────────────┐
│ [스타벅스 강남점 ▼]│  ← 매장 선택 드롭다운 (다중 매장)
│ [+ 매장 추가      ]│  ← 항상 노출 (오너만)
└────────────────────┘
```

**매장 추가 모달**
```
┌─────────────────────────────┐
│ 새 매장 추가            [×] │
├─────────────────────────────┤
│ 매장명 *                    │
│ [________________________]  │
│                             │
│ 사업자명 *                  │
│ [________________________]  │
│                             │
│ 사업자번호 *                │
│ [000  -  00  -  00000    ]  │
│                             │
│ 주소 *                      │
│ [________________________]  │
│ [상세주소                ]  │
│                             │
│ 매장 전화번호               │
│ [________________________]  │
│                             │
│ 담당자 휴대폰번호           │
│ [________________________]  │
│                             │
│ 업종                        │
│ [카페 ▼                  ]  │
│                             │
│         [취소] [매장 추가]  │
└─────────────────────────────┘
```

**동작**
- 저장 성공 → 매장 목록 갱신 → 새 매장을 `currentStoreId`로 자동 전환
- `+ 매장 추가` 버튼은 **OWNER 역할만** 노출 (MANAGER·STAFF는 미표시)

```
┌─────────────────────┐
│ ← 스케줄        [+] │
│ [주간] [월간]       │
│  < 04.14 - 04.20 >  │
├────┬────┬────┬──────┤
│    │ 월 │ 화 │ 수 ··│
├────┼────┼────┼──────┤

│10  │████│████│      │
│11  │    │████│████  │
│12  │    │    │████  │
│13  │    │    │      │

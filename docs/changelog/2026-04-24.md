### 모바일 홈 화면 출퇴근 체크 버그 수정

**문제 1: STAFF 권한으로 매장 전체 출근 API 조회 시 403 오류**
- 홈 화면이 `/stores/:id/attendance`(오너/매니저 전용)를 사용해 STAFF 로그인 시 403 반환
- `/me/attendance` 엔드포인트로 교체, 오늘 날짜 기준 로컬 필터링으로 변경

**문제 2: React Hooks 규칙 위반 — early return 위치 오류**
- `if (!currentStoreId) return <NoStoreScreen />` 가 `useQuery` / `useMutation` 호출 전에 위치
- `currentStoreId` 가 null ↔ 값 토글 시 "Rendered fewer hooks than expected" 오류 발생
- early return을 모든 hooks 호출 이후로 이동

**문제 3: axios 401 무한 refresh 루프**
- `/auth/refresh` 실패 시 동일 인터셉터가 다시 refresh를 재시도해 무한 루프 발생
- `original.url?.includes('/auth/refresh')` 조건으로 refresh 엔드포인트 재시도 차단

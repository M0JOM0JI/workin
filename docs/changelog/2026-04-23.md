## 변경사항

### ✨ 새 기능 (Features)

**API (NestJS)**
- `Store` 모델 필드 확장: `businessOwner`, `businessNumber`, `phone`, `mobilePhone` 컬럼 추가
- Prisma 마이그레이션: `20260423_add_store_business_fields`
- `CreateStoreDto` / `UpdateStoreDto` 확장 필드 반영
- `PATCH /stores/:id/staffs/:staffId` 엔드포인트 추가
  - `UpdateStaffDto`: `hourlyWage` (선택), `leftAt` (선택, null 허용)
  - OWNER/MANAGER 권한 검증 후 시급·퇴직일 업데이트
- `PATCH /stores/:storeId/attendance/:attendanceId` 엔드포인트 추가
  - OWNER/MANAGER가 clockIn/clockOut 직접 수정
  - `clockOut: null` 전송 시 퇴근 기록 제거 (근무 중 상태 복원)

**웹 대시보드 (Next.js)**
- **온보딩 페이지** `/onboarding/store`: 첫 매장 등록 전용 페이지
- **로그인·가입 후 리다이렉트** 개선: `stores.length === 0` 이면 `/onboarding/store`로 이동
- **사이드바 매장 추가 모달** 확장: 기존 3개 필드 → 전체 7개 필드
- **매장 설정 페이지** 확장: 사업자명·사업자번호·매장전화번호·담당자 휴대폰번호 필드 추가
- **직원 상세 모달**: 카드 클릭 → 이름·역할·이메일·전화·입사일 표시, 시급 수정, 퇴직 처리
- **출퇴근 수동 수정 모달**: 행 클릭 → KST 시간 입력, 출퇴근 역전 유효성 검사, 퇴근 제거 버튼
